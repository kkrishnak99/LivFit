LivFit - Fitness Advisor Android application

Livfit is a fitness advisor android application which shows recommended exercises and diet plans to the registered users based of their BMI which is calculated based on the information collected during signUp (Height, Weight and Gender).

Users have the option to browse through all the available exercises and diet plans without signing in to the application but it won't show recommended plans based on BMI.

Frontend : XML
Backend : Java
Database : SQLite
